Workflow

From feature request
to implementation plan.

Four steps from idea to code-ready plan.

1

Describe the feature

Enter a description or import a story from your task manager. Add specs, constraints, and project context.

2

Agents plan together

Tech Lead designs architecture, Developer creates implementation plan, QA produces test plan. If QA finds issues, Developer revises automatically.

3

Review and edit

Each agent's output in an editable view. Tweak the architecture, adjust the implementation plan — before any code is written.

4

Hand off to Claude Code

Click "Open Claude Desktop" or "Open in Terminal". Planning docs saved to your project, Claude Code implements step by step.
